Welcome to the CNICS Data Query Tool

CNICS is a consortium of 10 HIV clinics at academic medical centers. To improve HIV outcomes, we gather, use clinically, and share detailed and anonymous data reflecting context, treatment and outcomes in day-to-day care of people with HIV. Funded through the National Institute of Allergy and Infectious Diseases (NIAID), CNICS supports translational, clinical, and behavioral HIV research, and welcomes proposals from any researcher.

This data query system helps us implement NIH principles of making research data Findable, Accessible, Interoperable, and Reusable. We use the Leaf framework to allow high level exploration of CNICS data.

To explore CNICS data use username: test@example.com and password: testtest. This system is active under development, feel free to contact lober@uw.edu with feedback.

Welcome to the online demo of the CNICS Query Tool. As this is a demonstration version, we have temporarily disabled some security features in this version to allow anonymous logins. Anonymous logins are not a normal configuration for our systems, so you may experience some odd behavior, including unexpected logouts. Please bear with this temporary limitation.
